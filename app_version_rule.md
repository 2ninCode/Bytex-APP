# Bytex App Versioning Rule

Current Version: **V3.1**
Date: 2026-03-10

## Rule
The version follows the format: `V[Month].[UpdateNumber]`
- **Month**: The current month number (e.g., 3 for March).
- **UpdateNumber**: The sequential update number within that specific month.
